Este projeto de lista de tarefas, foi dealizado por mim , para poder ter em mãos um ferramenta básica, que eu possa fazer uma organização das minhas atividades diárias.
Este projeto me ajudou a praticar alguns conceitos que estava aprendendo em meu curso.
Algumas dos conceitos são, os de variáveis, constante, laços de repetição, integração com banco de dados MySQL, consultas e listagem dos registros destas consultas.
Este projeto ainda terá algumas atualizações e aperfeiçoamentos, tanto no back-end com no front-end.
